***Grumpy*** is a discord bot wrote in python with discord.py 2.6.4.

<ins>***Features:***</ins>
>- Get a role by message reaction.
>- Warn a guild member.
>- Report guild member.
>- Level system.
>- Mini games.
>- Ask the bot to said a joke.

<br/>

<ins>***Dependecies:***</ins>
>- [Python 3.10 or more](https://www.python.org/downloads/).
>- aiohttp
>- [discord.py==2.6.4](https://github.com/Rapptz/discord.py).
>- [python-dotenv](https://github.com/theskumar/python-dotenv).

<br/>

<ins>***Installation:***</ins>
1. Copy this folder to your machine.
2. Complete ```.env.example``` and rename it to ```.env```.
    - ```DISCORD_BOT_TOKEN``` The token of your bot available on discord developper portal.
    - ```TEST_GUILD_ID``` Your development server ID (not required).
    - ```CLIENT_ID``` Your client ID (not required).
    - ```INVITE_LINK``` The bot invitation link.
3. Install the dependencies:
    - ***Windows:*** Run ```install_grumpy.bat```
    - ***Linux:*** Run ```install_grumpy.sh```
4. Run the bot:
    - ***Windows:*** Run ```launch_grumpy.bat```
    - ***Linux:*** Run ```launch_grumpy.sh```
